About

프로젝트 소개

📖 프로젝트 개요

제목

“IT 직종과 인구 변화: 데이터로 보는 미래 전망”

목적

  • 사회과 AI융합교육을 위한 Quarto 웹 페이지 제작
  • 인구문제(저출산, 고령화, 인구절벽)와 IT 직종의 관계 탐구
  • 특성화고 학생들의 진로 설계 지원
  • 데이터 리터러시 및 데이터 기반 의사결정 역량 함양

🎯 학습 목표

지식·이해

  • 인구 구조 변화와 노동 시장의 관계 이해
  • IT 산업의 성장 배경과 전망 파악
  • KOSIS 데이터 활용 방법 습득

과정·기능

  • 데이터 수집, 분석, 시각화 능력 개발
  • Python/R을 활용한 데이터 처리
  • Quarto를 활용한 웹 문서 작성
  • 비판적 데이터 해석 능력

가치·태도

  • 데이터 기반 의사결정의 중요성 인식
  • 사회 문제에 대한 관심과 탐구 자세
  • 미래 지향적 진로 설계

📊 데이터 출처

모든 데이터는 KOSIS(국가통계포털)에서 수집했습니다.

사용한 주요 통계

통계 분류 통계명 기관
인구 장래인구추계 통계청
인구 시도/연령별 인구 통계청
고용 직업별 취업자 수 통계청(경제활동인구조사)
고용 산업별 취업자 수 통계청
교육 학령인구 교육부/한국교육개발원
사업 산업별 사업체 수 통계청(전국사업체조사)

데이터 수집 기간

  • 기준 데이터: 2015-2024년
  • 전망 데이터: 2025-2050년 (통계청 장래인구추계)

데이터 접근

  • KOSIS 웹사이트: https://kosis.kr
  • 데이터는 누구나 무료로 다운로드 가능
📌 데이터 신뢰성

본 프로젝트에서 사용한 모든 데이터는 정부 공식 통계로, 신뢰할 수 있는 출처입니다. 단, 미래 예측 데이터는 현재 추세를 바탕으로 한 전망이므로 실제와 다를 수 있습니다.

🛠️ 사용 기술

웹 퍼블리싱

  • Quarto: 문서 작성 및 웹사이트 생성
  • Markdown: 텍스트 작성
  • HTML/CSS: 디자인 커스터마이징

데이터 분석 및 시각화

  • Python: 데이터 처리 및 분석
    • pandas: 데이터 조작
    • plotly: 인터랙티브 시각화
    • scikit-learn: 간단한 예측 모델
  • R (선택적): 통계 분석

버전 관리

  • Git/GitHub: 프로젝트 관리 및 배포

💡 프로젝트 특징

1. 데이터 중심 접근

  • 주관적 의견이 아닌 객관적 데이터 기반
  • 실제 정부 통계 활용
  • 시각화를 통한 직관적 이해

2. 학생 관심사 연계

  • 취업이라는 현실적 주제로 시작
  • 특성화고 학생의 관점 반영
  • 구체적인 실천 방안 제시

3. 인터랙티브 학습

  • 클릭 가능한 그래프
  • 자가 진단 체크리스트
  • 개인화된 학습 경험

4. AI 융합

  • 데이터 예측 모델 활용
  • 코딩 실습 포함
  • 미래 역량 개발
